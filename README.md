# ykmc/atcoder

atcoderのコンテスト提出・過去問提出を管理するためのリポジトリ。2019年は2019ACを目標に。



## 言語とバージョン

- Python
  - 3.4.3
- Rust
  - 1.15.1



## ディレクトリ構造

```
 yyyy / mmdd_contestxxx / problem_yyyymmdd.ext
```

- yyyy, mmdd
  - コンテスト実施年, 月日
- contest, xxx
  - コンテスト名の省略表記, コンテスト番号
- problem
  - 問題番号(A-Z)
- yyyymmdd
  - submit年月日
- ext
  - ファイル拡張子
